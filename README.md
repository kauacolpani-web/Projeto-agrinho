# 🌳Agro forte, futuro sustentável: equilíbrio entre produção e meio ambiente🌎
🚜O agronegócio é muito importante para a sociedade, pois produz alimentos, gera empregos e ajuda na economia do país. Porém, para que o campo continue produzindo no futuro, é necessário cuidar do meio ambiente. Por isso, o tema do Agrinho 2026 mostra a importância de encontrar um equilíbrio entre a produção agrícola e a preservação da natureza.♻️

💧 Hoje existem várias formas de produzir sem causar tantos impactos ambientais. Os agricultores podem economizar água, proteger os rios, preservar áreas de vegetação e utilizar tecnologias que ajudam a reduzir o desperdício. Essas ações tornam a produção mais eficiente e sustentável. ♻️

🤖 A tecnologia também tem um papel importante. Máquinas modernas e sistemas inteligentes permitem que os produtores utilizem apenas a quantidade necessária de recursos, aumentando a produtividade e diminuindo os danos ao meio ambiente.

👨‍🌾 Além dos produtores, toda a sociedade pode contribuir para um futuro sustentável. Evitar desperdícios, consumir de forma consciente e valorizar produtos produzidos com responsabilidade são atitudes que ajudam a preservar os recursos naturais. 🌿

🌍 O equilíbrio entre produção e meio ambiente é essencial para garantir alimentos, desenvolvimento econômico e qualidade de vida para as próximas gerações. Quando o campo produz de maneira sustentável, todos são beneficiados. Assim, um agro forte não é apenas aquele que produz mais, mas também aquele que respeita a natureza e ajuda a construir um futuro melhor para todos. ✨

🖥️ Como o projeto foi desenvolvido

O projeto foi desenvolvido utilizando HTML, CSS e JavaScript. O HTML foi usado para estruturar o conteúdo da página, criando seções como o cabeçalho, a apresentação principal, as tecnologias do campo e os dados sobre o impacto da tecnologia no agronegócio. O CSS foi utilizado para personalizar o visual do site, definindo cores, fontes, espaçamentos, efeitos de sombra e o layout responsivo das informações. Já o JavaScript foi empregado para adicionar interatividade à página, tornando a navegação mais dinâmica e agradável.

🌱 Objetivo do projeto

O objetivo do site é apresentar como a tecnologia contribui para o desenvolvimento do agronegócio, destacando ferramentas como drones, inteligência artificial, sensores e máquinas autônomas. Essas tecnologias ajudam a aumentar a produtividade, reduzir desperdícios e promover práticas mais sustentáveis no campo.

🎨 Design do projeto

Foi utilizada uma identidade visual inspirada na natureza, com tons de verde e imagens relacionadas à agricultura, reforçando a conexão entre inovação tecnológica e preservação ambiental. Além disso, o site foi organizado em cartões informativos e seções bem distribuídas para facilitar a leitura e a compreensão do conteúdo.

📂 Estrutura dos arquivos

index.html → estrutura da página.
style.css → aparência e design do site.
script.js → funcionalidades e interações.
